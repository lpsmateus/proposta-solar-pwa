# ☀️ Proposta Solar PWA

Aplicativo web instalável (PWA) para geração de propostas comerciais de sistemas fotovoltaicos em **PDF**.  
Repositório oficial: [proposta-solar-pwa](https://github.com/lpsmateus/proposta-solar-pwa)

---

## 🚀 Como usar
1. Acesse: [https://lpsmateus.github.io/proposta-solar-pwa/](https://lpsmateus.github.io/proposta-solar-pwa/)
2. Preencha os dados do cliente e selecione o kit.
3. Clique em **Exportar PDF** para gerar a proposta (2 páginas).

---

## 📱 Instalar como App
- **Android (Chrome):** abra o link → menu ⋮ → **Adicionar à tela inicial**.  
- **iPhone (Safari):** abra o link → **Compartilhar → Adicionar à Tela de Início**.  

Depois disso, o app ficará disponível na tela inicial com ícone da sua logo e funcionará **offline**.

---

## 🛠️ Tecnologias
- **HTML + CSS + JavaScript**
- [html-to-image](https://github.com/bubkoo/html-to-image)
- [jsPDF](https://github.com/parallax/jsPDF)
- **PWA (Progressive Web App)** → manifest + service worker para cache e instalação offline

## 🔄 Atualizações
Se uma mudança não aparecer, aumente a versão do `CACHE` no `service-worker.js`  
(ex.: de `proposta-solar-v1` para `proposta-solar-v2`) e recarregue a página.
